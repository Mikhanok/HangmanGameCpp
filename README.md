# Console Hangman Game (Виселица)

Простая консольная игра «Виселица», написанная на языке C++. Игрок должен угадать случайное название фрукта на английском языке по буквам. С каждой ошибкой на экране дорисовывается виселица.

## 🚀 Особенности проекта

* **Кроссплатформенная очистка экрана:** Корректно очищает консоль как на Windows (`cls`), так и на macOS/Linux (`clear`).
* **Динамическая таблица:** Игровое поле автоматически подстраивается под длину загаданного слова.
* **Псевдографика:** Текстовая отрисовка виселицы в реальном времени (максимум 6 ошибок).
* **Случайный выбор слов:** База данных включает в себя популярные фрукты (apple, banana, mango и др.).

## 🛠️ Требования

* Компилятор с поддержкой стандарта **C++11** или выше (GCC, Clang, MSVC).
* Терминал или командная строка для запуска.

## 💻 Как запустить локально

### Через Терминал (macOS / Linux)
1. Откройте Терминал в папке с файлом проекта.
2. Скомпилируйте код:
   ```bash
   g++ -std=c++11 main.cpp -o hangman
   ```
3. Запустите игру:
   ```bash
   ./hangman
   ```

### Через Xcode (macOS)
1. Создайте новый проект типа **Command Line Tool**.
2. Вставьте этот код в файл `main.cpp`.
3. Нажмите комбинацию клавиш `Cmd + R` для сборки и запуска.

## 🎮 Игровой процесс

1. На экране отображается пустая таблица, количество ячеек совпадает с количеством букв в слове.
2. Введите букву в консоль и нажмите `Enter`.
3. Если буква угадана, она откроется на поле. Если нет — добавится элемент виселицы.
4. Игра заканчивается победой (`!!!YOU WON!!!`), если слово отгадано полностью, или поражением (`!!!GAME OVER!!!`), если совершено 6 ошибок.
